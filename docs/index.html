📋 安装说明

Android 手机安装步骤：

  1. 点击上方"下载 APK"按钮，下载安装包
  2. 在手机设置中开启"允许安装未知来源应用"
  3. 找到下载的 APK 文件，点击安装
  4. 按照提示完成安装
  5. 打开"课程小管家"，开始使用！
⚠️ 注意：

由于本 APP 未上架应用商店，安装时系统可能会提示"此应用可能有害"。这是正常提示，请点击"继续安装"或"仍要安装"。
