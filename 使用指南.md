## 📱 应用简介

**课程小管家** 是一款专为小学家长设计的课表管理工具，支持：

- 📅 周课表管理（支持 AB 周智能切换）
- 🎒 智能书包清单（根据当天课程自动生成）
- 📤 多种导出格式（图片/Excel/Word/Markdown）
- ⚙️ 灵活配置（上课时间、节数、周末设置）
- 📱 离线使用（数据本地存储，保护隐私）

## 📥 安装指南

### Android 手机安装步骤

1. **下载 APK**  
   访问 https://zouyuming-ai.github.io/course-manager/ 点击"下载 Android APK"按钮

2. **允许未知来源安装**  
   - 打开手机 **设置** → **安全** (或 **应用管理**)
   - 找到 **「允许安装未知来源应用」** 或 **「未知来源」**
   - 开启权限（不同品牌手机位置可能不同）

3. **安装 APK**  
   - 在文件管理器中找到下载的 `course-manager-v1.0.25.apk`
   - 点击安装，按照提示完成

4. **⚠️ 安全提示**  
   系统可能提示"此应用可能有害"，这是正常提示（因为未上架应用商店），点击 **「继续安装」** 或 **「仍要安装」**

### 卸载旧版本（如果之前安装过）

如果之前安装过 debug 版本，需要先卸载：

```bash
adb uninstall com.zouyuming.coursemanager
```

或者手动在手机上卸载：**设置** → **应用管理** → **课程管家** → **卸载**

## 🛠️ 开发指南

### 环境配置

**必要路径**（请根据你的实际环境修改）：

```properties
# 在 local.properties 中配置
sdk.dir=/path/to/your/android-sdk
```

**编译命令**（请根据你的实际路径修改）：

```bash
# 设置环境变量（示例，请根据你的环境修改）
export PATH="$PATH:/path/to/flutter/bin"
export ANDROID_HOME="/path/to/android-sdk"
export ANDROID_SDK_ROOT="$ANDROID_HOME"
export JAVA_HOME="/path/to/jdk"
export PATH="$JAVA_HOME/bin:$ANDROID_HOME/cmdline-tools/latest/bin:$ANDROID_HOME/platform-tools:$PATH"

# 进入项目目录
cd /path/to/course_manager

# 编译 Release APK
flutter build apk --release

# APK 输出位置
# build/app/outputs/flutter-apk/app-release.apk
```

### 版本号更新流程

每次发布新版本需要同步更新 3 个地方：

1. **`pubspec.yaml`** (第 1 行)：
   ```yaml
   version: 1.0.26+26
   ```

2. **`lib/utils/app_version.dart`**：
   ```dart
   class AppVersion {
     static const String version = 'v1.0.26';
     static const int buildNumber = 26;
     static const String fullVersion = 'v1.0.26 (26)';
   }
   ```

3. **更新日志** (`CHANGELOG.md`)：
   ```markdown
   ## v1.0.26 (2026-07-10)
   - 新功能说明
   ```

### Release 签名配置

**Keystore 位置**：`android/keystore/release-key.jks`  
**配置文件**：`android/keystore.properties`

⚠️ **重要**：不要提交 `keystore.properties` 和 `*.jks` 到公开仓库（已加入 `.gitignore`）

签名密码请妥善保管，不要公开。

## 📤 发布流程

### 1. 更新版本号

按照上述「版本号更新流程」更新 3 个文件

### 2. 编译 APK

```bash
cd /path/to/course_manager
flutter build apk --release
```

### 3. 复制 APK 到桌面


访问 https://github.com/zouyuming-ai/course-manager/releases/new

- 更新下载链接
- 更新更新日志

然后推送到 GitHub（会自动更新分享页面）

### 6. 推送代码

```bash
cd /path/to/course_manager
git add .
git commit -m "release: v1.0.26"
git push origin main
```

## 🐛 常见问题

### 1. 安装失败："签名冲突"

**原因**：之前安装过 debug 版本，签名不同  
**解决**：卸载旧版本，重新安装

### 2. 图片导出只有表头

**原因**：已修复（v1.0.23）  
**解决**：更新到最新版

### 3. AB 周显示不正确

**原因**：已优化（v1.0.25）  
**解决**：更新到最新版，确保有 AB 周课程时才会显示相关 UI

### 4. 课程名显示不完整

**原因**：已修复（v1.0.25）  
**解决**：5 字以上会自动换行

### 5. 编译失败

**排查步骤**：
1. 检查 `local.properties` 中的 SDK 路径是否正确
2. 运行 `flutter doctor` 检查环境
3. 删除 `build/` 目录重新编译：`flutter clean && flutter build apk --release`
