# 应用截图指南

## 📸 截图要求（小米应用商店）

- **数量**：至少 3 张，建议 5-8 张
- **尺寸**：1280x720 或 1920x1080（16:9 或 9:16）
- **格式**：PNG 或 JPG
- **内容**：展示应用核心功能，无个人信息

---

## 📱 推荐截图内容

### 1. 课表主页（最重要）
**内容**：展示一周课程表，色彩丰富
**说明**：让用户一眼看懂应用的核心功能
**操作**：
1. 打开应用
2. 进入课表页面
3. 截图

### 2. 书包清单页
**内容**：展示某天的书包物品清单
**说明**：展示核心差异化功能
**操作**：
1. 点击某天
2. 查看书包清单
3. 截图

### 3. 添加课程页
**内容**：展示添加课程的界面
**说明**：展示易用性
**操作**：
1. 点击编辑课表
2. 展示添加课程界面
3. 截图

### 4. 导出分享功能
**内容**：展示导出选项（图片/Excel/Word）
**说明**：展示实用功能
**操作**：
1. 点击导出按钮
2. 展示导出选项
3. 截图

### 5. 设置页面
**内容**：展示设置选项
**说明**：展示个性化能力
**操作**：
1. 进入设置
2. 截图

### 6. 提醒设置页
**内容**：展示提醒配置
**说明**：展示智能化功能
**操作**：
1. 进入提醒设置
2. 截图

---

## 📷 截图步骤

### 在真机上截图
1. 安装 `app-release.apk` 到手机
2. 依次进入上述页面
3. 使用系统截图功能：
   - **小米手机**：电源键 + 音量减键
   - **大部分 Android**：电源键 + 音量减键
4. 将截图传输到电脑
5. 放到 `app_store_assets/xiaomi/screenshots/` 目录

### 截图注意事项
- 确保内容完整，无截断
- 确保无个人信息（姓名、学校名等）
- 确保界面干净，无通知栏干扰
- 可以稍微美化（添加手机边框等，可选）

---

## 🎨 截图优化（可选）

如果需要更专业的截图，可以：
1. 使用手机边框模板（网上搜索"手机截图模板"）
2. 添加简短文字说明（用 Photoshop 或在线工具）
3. 统一风格（相同的边框、相同的文字位置）

---

## 📤 传输截图到电脑

### 方法 1：USB 数据线
1. 连接手机和电脑
2. 在手机上选择"文件传输"模式
3. 复制截图文件到电脑

### 方法 2：网络传输
- 使用 QQ/微信 文件传输助手
- 使用小米云同步
- 使用邮箱发送

### 方法 3：adb 命令（如果手机已连接）
```bash
adb pull /sdcard/Pictures/Screenshots/ ./app_store_assets/xiaomi/screenshots/
```

---

**准备完成后**，将截图文件放到：
`course_manager/app_store_assets/xiaomi/screenshots/`

- `screenshot_04_导出分享.png`
