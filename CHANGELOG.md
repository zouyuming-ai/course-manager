# 课程小管家 - 版本变更记录

## v1.0.0 (1) - 2026-07-02 🎉 首个正式版

### 核心功能
- **课表管理**：可视化课表展示，支持按周查看
- **书包清单**：为每天配置书包携带物品，帮助家长和孩子整理书包
- **提醒通知**：课前提醒和书包整理提醒，支持自定义提醒时间
- **课表导出**：支持导出为图片、Excel、Word、Markdown 格式，方便分享和打印
- **预置课表**：内置多套常见课表模板，一键导入使用
- **科目颜色自定义**：为不同科目设置颜色，课表更直观
- **时间点配置**：自定义上下课时间，适配不同学校作息

### 技术优化
- **移除 OCR 功能**：简化应用，专注核心课表管理体验
- **应用图标优化**：蓝底白线书包 + 课表网格设计，简洁易识别
- **性能优化**：移除不必要的依赖，APK 体积减小

## v1.0.12 (12) - 2026-07-01

### 新功能
- **时间点配置系统（P0）**：
  - 新增 `TimeSlotProvider`，支持按学期配置上下课时间
  - 新增时间点设置页面 (`/time-slots`)，支持新增/编辑/删除/重置默认
  - `TimeSlot` 模型新增 `label` 字段，支持自定义节次标签
  - 课表页面现在从 `TimeSlotProvider` 读取时间节点，不再硬编码
- **Word 真 .docx 导出（P1）**：使用 `docx_dart` 包生成真正的 .docx 文件，不再用 HTML 伪装
- **假期日历结构化数据（P2）**：新增 `holiday_data.dart`，支持 2025-2028 年法定假日数据
  - 2025/2026 年数据完整（国务院已发布）
  - 2027/2028 年预留空数据模板

### 改进
- **OCR 智能网格解析（P1）**：利用 `TextBlock.boundingBox` 坐标做网格解析，提高课表识别准确率
  - 对 Y 坐标聚类识别行/节次，对 X 坐标聚类识别列/星期
  - 在网格单元格内匹配科目名，减少位置推断错误
  - 保留降级逻辑，兼容格式特殊的课表图片
- **导出时带上时间节点**：Excel/Word/Markdown 导出现在在节次标签中显示时间段（如 "第一节 (8:30-9:10)"），若已配置时间点
- **假期日历支持多年度**：从硬编码 2026 年数据改为结构化数据，支持 2025-2028 年

### 设置页
- 新增"时间点配置"入口

## v1.0.11 (11) - 2026-06-30

### 新功能
- **推送通知**：课程提醒和书包提醒现在会实际推送本地通知（每天早上提醒当天课程、每晚提醒书包准备、自定义提醒按时推送）
- **导出功能重构**：告别纯文本复制，新增 5 种实用导出格式
  - 保存为图片 → 保存到相册，可直接分享到微信
  - 导出 Excel (.xlsx) → 可用 Excel/WPS 编辑
  - 导出 Word (.doc) → 可用 Word/WPS 查看（HTML格式表格）
  - 导出 Markdown (.md) → 适合文本编辑和分享
  - 分享 → 通过微信/QQ等系统分享面板分享课表图片
- **OCR导入课表**：拍照或从相册选择课表图片，自动识别课程内容，半自动确认后批量导入
